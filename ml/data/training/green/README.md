# Green Talisay Fruit Training Images

Place images of **GREEN (immature)** Talisay fruits in this folder.

## Image Guidelines:
- Clear, well-lit photos
- Fruit should be the main subject
- Various angles welcome (front, side, top)
- Minimum resolution: 224x224 pixels (recommended: 512x512+)
- Formats: JPG, PNG, JPEG

## Characteristics of Green Talisay:
- Predominantly green skin color
- Immature/unripe stage
- Firm flesh
- Lower oil content (~45-49%)

## Naming Convention (optional):
- `green_001.jpg`
- `green_002.jpg`
- etc.

Aim for at least 50-100 images per category for basic training.
